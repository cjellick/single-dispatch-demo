PEP 443 Demo
===============
Single-dispatch generic functions
--------------

You'll need Python 3.4. Use this command to create a 3.4 virtual environment:
```
virtualenv -p /usr/local/bin/python3.4 venv/
. venv/bin/activate
```

To run the example:
```
python main.py
```
